# ChessElo

SwiftUI iOS app for tracking chess ratings, stats, and leaderboards.

## Features
- Authentication (Supabase)
- Player stats and history
- Leaderboard
- Widgets

## Tech
- Swift / SwiftUI
- Supabase (Auth + Database)
- WidgetKit

## Requirements
- Xcode 15+
- iOS 17+
